# Sistema de Gerenciamento de Usuários em Java

Projeto simples feito em Java para praticar lógica de programação e POO.

## Funcionalidades

- Cadastrar usuários
- Listar usuários
- Buscar usuário
- Remover usuário

## Como executar

Abra o projeto em uma IDE (Eclipse ou IntelliJ) e execute a classe Menu.java.
